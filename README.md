## 📋 Структура проекта

### Основной файл

- **`index.html`** — полный лендинг-портфолио в одном файле (10 секций)

### Активы

- `SamH.png` — фото для hero-секции
- `Sam.png` — фото для секции "Обо мне"
- `my-landing-QR.png` — QR-код на сайт

### Документация

- `project.md` — описание проекта, целевая аудитория, дизайн-система
- `brief.md` — подробное техническое задание
- `research.md` — анализ конкурентов и рекомендации
- `CLAUDE.md` — правила работы с проектом
- `PLAN.md` — архив планов и решений

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере
2. Все стили и скрипты встроены в файл (нет зависимостей)
3. Измените контент в текстовых секциях по необходимости

### Локальная разработка

```bash
# Просмотр в браузере
open index.html

# Или используй live-server
npx live-server
```

## 🎭 Дизайн-система

### Цветовая палитра

```
Фон:           #F8FAFF, #EEF2FF
Акцент 1:      #6366F1 (Индиго)
Акцент 2:      #8B5CF6 (Фиолетовый)
Акцент 3:      #F43F5E (Розовый)
Текст:         #0F172A
Второстепенный: #64748B
```

### Шрифты

- **Заголовки:** Space Grotesk
- **Текст:** Inter
- **Код:** JetBrains Mono

### Компоненты

- Белые карточки с тенями и скруглёнными углами
- Анимированные градиентные заголовки
- Hover-эффекты (lift, magnetic buttons)
- Scroll-reveal анимации
- Кастомный курсор (dot + follower)

## ✨ Ключевые особенности

### 3D Нейросеть в Hero

- 58 узлов, реагирующих на движение мыши
- Анимированные импульсы между узлами
- Адаптивно скрывается на мобильных устройствах
- **Исправлено (апр 2026):** добавлена обработка ошибок в рендеринге

### Адаптивность

- Полностью отзывчивая (320px–4K)
- Мобильное меню с smooth animations
- Оптимизирована для всех устройств

### Производительность

- Без внешних библиотек (только HTML, CSS, JS)
- Встроенные SVG-иконки
- Минимальный размер
- Быстрая загрузка

## 🔧 Разработка

### Правила проекта

Смотрите `CLAUDE.md` для полного списка правил:

- Одностраничный лендинг (все в `index.html`)
- Чистый CSS/JS (без фреймворков)
- Минималистичный, современный дизайн
- Мобайл-first подход
